# Getting Started with Unhireable

Welcome to **Unhireable** - Your Neural Career System. This guide will help you get started with automating your job search.

## What is Unhireable?

Unhireable is a desktop application that helps you:
- **Automatically find jobs** from multiple sources
- **Generate tailored resumes and cover letters** using AI
- **Track your applications** and interviews
- **Match with jobs** that fit your profile
- **Automate job applications** with browser automation

## Installation

### Prerequisites

- **macOS, Windows, or Linux** (desktop application)
- **Internet connection** (for job scraping and AI features)

### Download and Install

1. Download the latest release from [GitHub Releases](https://github.com/DaurenNope/jobEz/releases)
2. Install the application:
   - **macOS**: Open the `.dmg` file and drag to Applications
   - **Windows**: Run the `.msi` installer
   - **Linux**: Extract the `.AppImage` and make it executable

### First Launch

1. Launch Unhireable from your Applications folder
2. The app will guide you through initial setup:
   - Create a master password (for secure credential storage)
   - Set up your user profile
   - Configure AI API keys (optional, for enhanced features)

## Initial Setup

### 1. Create Your Profile

Your profile is essential for job matching and document generation:

1. Go to **Settings** → **Profile**
2. Fill in your information:
   - **Personal Info**: Name, email, phone, location, LinkedIn, GitHub
   - **Skills**: Technical skills, soft skills, years of experience
   - **Experience**: Work history with descriptions
   - **Education**: Educational background
   - **Projects**: Notable projects

3. Click **Save Profile**

**Tip:** The more complete your profile, the better job matches you'll get!

### 2. Configure AI Provider (Optional)

For AI-powered document generation:

1. Go to **Settings** → **Credentials**
2. Add your AI provider API key:
   - **OpenAI**: Get key from [platform.openai.com](https://platform.openai.com)
   - **Anthropic**: Get key from [console.anthropic.com](https://console.anthropic.com)
   - Or use other supported providers

3. Click **Save**

**Note:** AI features work without API keys, but with limited functionality.

### 3. Configure Email Notifications (Optional)

To receive email alerts:

1. Go to **Settings** → **Email Notifications**
2. Configure SMTP settings:
   - **Gmail**: Use app-specific password
   - **Outlook**: Use app password
   - **Custom SMTP**: Enter your server details
3. Click **Test Connection** to verify
4. Enable notification types you want

## Your First Job Search

### Step 1: Search for Jobs

1. Go to **Jobs** page
2. Enter a search query (e.g., "React developer")
3. Select job sources (LinkedIn, Wellfound, etc.)
4. Click **Scrape Jobs**

**Note:** Scraping may take a few minutes depending on sources selected.

### Step 2: Calculate Match Scores

1. After jobs are loaded, click **Calculate Match Scores**
2. The app will match each job against your profile
3. Jobs are sorted by match score (highest first)

### Step 3: Generate Documents

1. Select a job with a high match score
2. Click **Generate Resume** or **Generate Cover Letter**
3. Choose a template
4. Review and customize the generated document

### Step 4: Apply to Jobs

1. Click **Apply** on a job
2. Choose application method:
   - **Manual**: Download documents and apply manually
   - **Automated**: Use browser automation (if supported)
